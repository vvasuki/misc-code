procedure to use javadoc:
This example uses -sourcepath so javadoc can be run from any directory and
-subpackages (a new 1.4 option) for recursion. It traverses the subpackages of
the java directory excluding packages rooted at java.net and java.lang. Notice
this excludes java.lang.ref, a subpackage of java.lang).
javadoc -d /home/html -sourcepath /home/src -subpackages java -exclude
java.net:java.lang
